# Dispo-Kalender Demo (HTML/CSS/JS)

Kleines Frontend-Prototyping für einen Betrieb mit ca. 10 Mitarbeitenden.

## Features
- Wochenansicht mit KW-Anzeige
- Zeitslots für Disposition pro Tag
- Absenzen erfassen (Ferien, Feiertag, Militär, Unfall, Krankheit)
- Fake-Daten als Startpunkt
- Speicherung lokal im Browser (`localStorage`)

## Start
Einfach `index.html` im Browser öffnen.
